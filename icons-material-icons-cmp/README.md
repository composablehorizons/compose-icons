# Material Icons icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Material Icons icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-material-icons-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Material Icons` object:

```kotlin
Image(Material Icons.Icon, contentDescription = null)
```
