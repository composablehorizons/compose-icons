# Feather icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Feather icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-feather-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Feather` object:

```kotlin
Image(Feather.Icon, contentDescription = null)
```
