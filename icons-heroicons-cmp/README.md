# Heroicons icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Heroicons icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-heroicons-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Heroicons` object:

```kotlin
Image(Heroicons.Icon, contentDescription = null)
```
