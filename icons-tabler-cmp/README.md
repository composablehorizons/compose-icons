# Tabler icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Tabler icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-tabler-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Tabler` object:

```kotlin
Image(Tabler.Icon, contentDescription = null)
```
