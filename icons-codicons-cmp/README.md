# Codicons icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Codicons icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-codicons-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Codicons` object:

```kotlin
Image(Codicons.Icon, contentDescription = null)
```
