# Bootstrap outline icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Bootstrap outline icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-bootstrap-outline-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Bootstrap` object (Outline variant):

```kotlin
Image(Bootstrap.Outline.Icon, contentDescription = null)
```
