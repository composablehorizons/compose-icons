# Tabler filled icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Tabler filled icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-tabler-filled-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Tabler filled` object (Filled variant):

```kotlin
Image(Tabler filled.Filled.Icon, contentDescription = null)
```
