# Material Icons Filled icons for Android

An Android icon library with all available Material Icons Filled icons as XML drawables.

## Installation

Add the library to your app's `build.gradle.kts`:

```kotlin
dependencies {
    implementation("com.composables:icons-material-icons-filled-android:1.0.0")
}
```

## Basic usage

```kotlin
Icon(
    painterResource(R.drawable.material-icons_ic_icon_filled),
    contentDescription = null
)
```
