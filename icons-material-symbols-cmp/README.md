# Material Symbols icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Material Symbols icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-material-symbols-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Material Symbols` object:

```kotlin
Image(Material Symbols.Icon, contentDescription = null)
```
