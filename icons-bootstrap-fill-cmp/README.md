# Bootstrap Fill icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Bootstrap Fill icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-bootstrap-fill-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Bootstrap` object (Fill variant):

```kotlin
Image(Bootstrap.Fill.Icon, contentDescription = null)
```
