# Radix icons for Compose Multiplatform

A Compose Multiplatform icon library with all available Radix icons.

## Installation

```kotlin
dependencies {
    implementation("com.composables:icons-radix-cmp:1.0.0")
}
```

## Basic usage

All icons are accessible via the `Radix` object:

```kotlin
Image(Radix.Icon, contentDescription = null)
```
